# 🧮 Calculadora em JavaScript

Uma calculadora simples desenvolvida com **HTML, CSS e JavaScript**, com foco no aprendizado de programação web.

---

## 🚀 Funcionalidades

* Botões numéricos (0 a 9)
* Operações básicas:

  * ➕ Adição
  * ➖ Subtração
  * ✖️ Multiplicação
  * ➗ Divisão
* Botão de igual (=) para calcular o resultado

---

## 🖥️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📂 Estrutura do projeto

```
Calculadora/
│
├── index.html
├── style.css
└── script.js
```
